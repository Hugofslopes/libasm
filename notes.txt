INSTRUCTIONS 
Arithmetic
ADD R1,R2 adds the value from R2 to R1
SUB  R1, R2 subtracts are two from R1
MUL R1, R2
DIV R1, R2

LOgic
AND R1, R2 bitwise and 
OR R1, R2 bitwise or
XOR R1, R2 exclusive or
NoT R1 change the 1 to 0

Control
JMP LABEL JUmps to certain part of the code
CMP R1, R2 compares the values in R1 with R2
    JE jumps if they are equal 
    JNE Jumps it they are different

MOV R1, R2 moves data from R2 to r1
PUSH R1 pushes R1 to the stack 
POP R1 retrieves R1 from the stack

SHL R1,1 Shift bits left , multiplies the value by 2
SHR R1,1 Shift bits right, divides the value by 2
ROL R1,1 Rotate the bits to the left
ROR R1,1 Rotate the bits to the right

Fetch Decoding and executing 
Retrieves the instruction from the program

received instruction
